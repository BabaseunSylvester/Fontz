# Fontz 🅰️

A sleek Android app that transforms your text into a variety of Unicode font styles — bold, italic, cursive, monospace, and more — with a single tap to copy your preferred style to the clipboard.

---

## ✨ Features

- **Real-time text transformation** — type once, see all styles instantly
- **Multiple Unicode styles** — bold, italic, bold italic, cursive, double-struck, monospace, fraktur, and more
- **One-tap copy** — tap any style to copy it directly to your clipboard
- **Clean, minimal UI** — built with Jetpack Compose and Material 3

## 🚀 Getting Started

### Prerequisites
- Android Studio Hedgehog or later
- JDK 17+
- Android SDK 21+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BabaseunSylvester/Fontz.git
   ```

2. Open the project in Android Studio.

3. Let Gradle sync complete.

4. Run on an emulator or physical device (Android 5.0+).
